▸Overview

Pratibimb is a comprehensive, professional-grade system for converting English text to Grade 1 Unicode Braille. The name "Pratibimb" means "reflection" in Hindi, symbolizing our mission to create a true reflection of the digital text world in Braille format.

  • Professional Quality: Industry-standard Grade 1 Braille conversion following UNESCO standards

Parameter Description Default Options

tab_width Spaces per tab (regular) 4 2, 4, 8
line_length Characters per line (embosser) 40 30-42
page_length Lines per page (embosser) 25 20-27
include_page_numbers Add page numbers true true, false

🔧Troubleshooting

❌Common Issues

📁File Not Found

Error: ❌ Error: Input file 'file.txt' not found!

Solution: Check file paths in config.json and ensure files exist.

🔤Encoding Errors

Error: Character encoding issues during conversion

Solution: Try different encodings (utf-8, latin1) in configuration.

🖥️GUI Won't Start

Error: GUI application fails to launch

Solution: Ensure tkinter is installed (usually included with Python).

📦Batch Processing Fails

Error: Some files in batch couldn't be processed

Solution: Check file permissions and glob patterns.

🖨️Embosser Issues

📏Validation Fails

❌ LINE LENGTH ERRORS Page 1, Line 5: 42 chars (expected 40)

Solution: Rerun the formatter - it handles word wrapping automatically.

🖨️Embosser Won't Print

  1. Check file extension: Use .brl extension
  2. Verify encoding: Ensure UTF-8 encoding
  3. Test small file: Try a simple 1-page document first
  4. Check embosser settings: Verify embosser expects 40x25 format

💡Best Practices

  • Start Small: Test with 1-2 pages first
  • Validate Always: Never skip validation step
  • Keep Originals: Preserve source text files
  • Document Settings: Note any custom configurations
  • Regular Testing: Periodically verify embosser compatibility

📞Getting Help

  1. Check the error message details
  2. Verify file paths and permissions
  3. Test with simple text first
  4. Review configuration settings
  5. Check system requirements
